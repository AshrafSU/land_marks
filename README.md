# land_marks

This project is a SwiftUI-based iOS application that showcases various landmarks.

## Features

- List of landmarksviews
- Load data from a local JSON file
- Manage and display data using SwiftUI

## Requirements

- Xcode 12 or later
- iOS 14.0 or later
- Swift 5.3 or later
